# Android-adderss-book
课设大作业  
### 功能：  
1.  获取本机所有通话信息，并分类显示：  
a)	所有拨出电话  
b)	所有已接电话  
c)	所有未接电话  
2.	统计所有通话信息  
a)	获取通话时长最长的记录  
b)	获取通话次数最多的电话号码  
c)	获取最近5个通话记录  

---------
**模拟机：Pixel 3a XL API 29*

   > 优化布局，对较新的记录增加颜色，突出显示。  

2. 修改按键为：读取记录、最多、最久
